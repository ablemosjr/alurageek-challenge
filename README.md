<img src="https://github.com/ablemosjr/portfolio/assets/89210954/12db4797-195d-4c8e-90ef-fad6c4356080" width="100%" />

<p>  
Este desafio foi proposto pela [Oracle Next Education](https://www.oracle.com/br/education/oracle-next-education/) em parceria com a [Alura](https://www.alura.com.br/), com o objetivo de proporcionar uma experiência próxima à vivenciada por programadores front-end em seu cotidiano. A ideia é criar interfaces funcionais e desenvolver um raciocínio lógico voltado para a resolução de problemas.

O foco principal é praticar a criação de um <b>CRUD</b>, concentrando-se nas etapas de Criação, Leitura e Exclusão de dados.
</p>

## :computer: Funções

<h4>Botão de navegação</h4>

![repository button](https://github.com/ablemosjr/eCommerce-challenge/assets/89210954/fecfa1fc-eb76-40c0-9902-840bd0c75b45)
- Redireciona para este repositório
<br>

<h4>Botão Adicionar Produto</h4>

![add new product button](https://github.com/ablemosjr/eCommerce-challenge/assets/89210954/e677f16e-d019-4303-b1bd-b66e4edb7607)
- Abre modal para adicionar produto ao banco de dados
<br>

<h4>Cartão do Produto</h4>

![card product](https://github.com/ablemosjr/eCommerce-challenge/assets/89210954/e4e4f621-ad84-4958-ab8c-ce8857db6450)
- Cartão de demonstração do produto
	- Contém:
		- Código do produto 
		- Imagem do produto
		- Nome do produto
		- Preço do produto
		- Botão de exclusão
<br>

<h4>Botão Excluir Produto</h4>

![delete button](https://github.com/ablemosjr/eCommerce-challenge/assets/89210954/9e9e43cd-c999-4c50-9df2-e3f51ebf7b85)
- Exclui produto do banco
<br>

<h4>Clique na Imagem</h4>

![open update modal image button](https://github.com/ablemosjr/eCommerce-challenge/assets/89210954/e2a56ffd-e337-4696-bddf-59f157b20bae)
- Abre modal para alterar informações do produto no banco de dados
<br>

<h4>Modal (Adicionar / Alterar produto)</h4>

![modal model](https://github.com/ablemosjr/portfolio/assets/89210954/a55e0988-eb34-4360-a58a-d1ecef93d26d)
- Modal aberto
<br>

<h4>Botão Guardar</h4>

![save/update product](https://github.com/ablemosjr/portfolio/assets/89210954/ad9bb384-78dc-4836-80d8-7bede942bc1a)
- Botão para salvar e/ou alterar produto (dependendo do modal aberto)
<br>

<h4>Botão Limpar</h4>

![clean fields](https://github.com/ablemosjr/portfolio/assets/89210954/676fe686-d47b-4238-835a-0c81b52061c0)

<h4>Botão Fechar Modal</h4>
